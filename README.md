# Design Principles for Scaling Agents Without Sprawl

Twelve principles for governing AI agents at enterprise scale — from defining outcomes to retirement.

## Files

- **design-principles-v2.md** — The principles (vendor-neutral)
- **applied-to-snowflake.md** — Mapping each principle to Snowflake features

## Quick reference

| Phase | Principles |
|---|---|
| Define it | P1 Outcomes, P2 Identity, P3 Front door |
| Build it | P4 Specs as code, P5 Reuse, P6 Vocabulary |
| Run it | P7 Least privilege, P8 Gates, P9 Trust |
| Keep it | P10 Observability, P11 Lifecycle, P12 Pave the road |
